Configuration and settings
--------------------------

Configuration and Settings describe the overall configuration of the
Sitecore system, implementation or configuration of specific features or
functionalities. These can take many forms, such as .config files or
data in Sitecore, and can be defined by the installed software such as
Sitecore or 3\ :sup:`rd` party modules or by the business logic in the
solution modules.
